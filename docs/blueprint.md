# **App Name**: Roshan Chat

## Core Features:

- User Authentication: Secure user sign-up/sign-in with Google and Gmail integration.
- Real-time Chat: Send and receive text messages in real-time.
- Multimedia Support: Send and receive image, video, and audio messages.
- HD Media File Integration: Integration with HD media file formats.
- Notifications: Get real-time notifications for new messages.
- Chat History: Maintain a history of past chats.
- AI-Powered Reply Suggestions: Use an AI tool to suggest appropriate quick replies based on message content.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) to convey innovation and beauty.
- Background color: Light purple (#F3E5F5), almost white, for subtlety and to highlight content.
- Accent color: Electric purple (#8E24AA), slightly different from the primary color, but still an analogous color, with boosted brightness and saturation for strong contrast.
- Headline font: 'Poppins', a geometric sans-serif font.
- Body font: 'PT Sans', a humanist sans-serif font, used for longer passages of text.
- Use crisp, modern vector icons for intuitive navigation.
- Clean, modern layout with an intuitive user interface for seamless navigation.
- Subtle, elegant transitions and animations to enhance user experience.
